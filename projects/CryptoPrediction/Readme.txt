Cryptopred
==========

This readme provides an overview on the Cryptopred related files and libraries

FILES PROVIDED
==============

This project includes the following files

1. CryptoPredv0.7.ipynb: Includes the Python Jupyter notebook code

2. ML Capstone - Proposal.pdf: Proposal for designing and implementing the solution

3. ML Capstone - Report v0.7.pdf: Implementation details for the machine learning model

4. dfcc.csv: Comma separated file including the training and test data

SETUP
=====

To setup the provided solution, we just need to run the Python Jupyter notebook on a PC with Internet connection as it will download the altcoin prices directly from cryptocompare.com website
